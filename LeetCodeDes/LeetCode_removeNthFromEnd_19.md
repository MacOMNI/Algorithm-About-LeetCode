# LeetCode No.19 [RemoveNthFromEnd](https://leetcode.com/problems/remove-nth-node-from-end-of-list/)

### 题目描述

给定一个链表，删除链表的倒数第 *n* 个节点，并且返回链表的头结点。

**示例：**

```
给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
```

### 题目解析
关键因素分析:


时间复杂度：O(n)
空间复杂度：O(n)
